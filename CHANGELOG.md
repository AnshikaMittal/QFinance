# Changelog

All notable changes to QuickFinance will be documented in this file.

## [0.1.0] - 2026-03-25

### Added
- Initial project scaffolding with React + Vite + TypeScript
- Dexie.js database layer with core schema
- PWA configuration with offline support
- CI/CD pipeline (GitHub Actions)
- Security scanning workflow
- Modular feature-based architecture
- Shared UI component library (Button, Card, Modal, Input, Badge, ProgressBar)
- Default spending categories with auto-categorization keywords
- Mobile-first responsive layout with bottom navigation
- Project documentation (README, Architecture, Agents, Contributing)
